Le Web

Historique

1965 :
Ted Nelson invente le concept (et le terme) d’hypertexte.
1989 :
Tim Berners-Lee propose au CERN (Centre Européen de Recherche Nucléaire) un système utilisant l’hypertexte pour faciliter la diffusion et le partage des informations entre les physiciens. Il invente l’URL (adresse d’un document), le HTML (langage d’écriture de document ) et le HTTP (protocole d’échange de fichier). C’est la naissance du Web.
1993 :
  • Le CERN annonce que la technologie du Web sera gratuite et libre de droit.
  • Marc Andreessen invente Mosaic, le premier navigateur grand public. Il est gratuit (pour un usage non commercial), doté d’une interface graphique (à opposer aux programmes en lignes de commande) et permet de visualiser des images dans le texte d’une page.
1994 :
Création du W3C (World Wide Web Consortium) pour standardiser les technologies du Web.
1995 :
mise à disposition de technologies pour le développement de site Web interactif (langage JavaScript) et dynamique (langage PHP) ;
2001 :
standardisation des pages grâce au DOM (Document Object Model : interface de programmation pour les documents HTML) ;
2010 :
mise à disposition de technologies pour le développement d’applications sur mobiles.

Hyperlien, hypertexte, hypermédia

  • Un hyperlien est un élément d'un document numérique dont l'usage renvoie vers une autre partie du document ou vers un autre document ou d'autres ressources (mail par exemple).
  • Un hypertexte est un texte, sous forme de fichier numérique, contenant au moins un hyperlien.
  • On élargit la notion d'hypertexte à celle d'hypermédia : une ressource numérique contenant un ou plusieurs médias (texte, image, audio, vidéo, animation, ...) et au moins un hyperlien.
  • Un hypermédia ou un hypertexte désigne aussi un ensemble de ressources numériques reliées par des hyperliens. Lire ce document dans lequel hypertexte et hypermédia sont synonymes.

Le Web

Le Web (ce qui signifie la « toile ») est l’abréviation du World Wide Web (la « toile d’araignée mondiale ») abrégé aussi en WWW.

C'est un système hypertexte (ou plutôt hypermédia) fonctionnant sur Internet.
Autrement dit, le Web est l’ensemble des ressources numériques (pages HTML, images, fichiers sons, fichiers vidéos, mail, ...) reliées par des hyperliens et accessibles sur des machines du réseau Internet. L’utilisateur peut alors naviguer entre ces ressources par l’intermédiaire des liens.

Attention : le Web n’est pas Internet.
Internet est le réseau mondial formé des divers réseaux informatiques, c'est un ensemble d'ordinateurs, de câbles et de liaisons wi-fi, de protocoles permettant à ces éléments de partager des informations, ...
Divers services se sont développés grâce à Internet, comme le mail (la messagerie életronique), le ftp (le téléchargement de fichiers), les forums de discussions (newsgroups ou chat irc).
Le Web n'est qu'un de ces services alors qu'Internet est le réseau informatique sur lequel sont stockées les ressources du Web et par lequel on y accède.
Pour faciliter le partage de ces ressources, le consortium mondial W3C (World Wide Web Consortium) recommande des normes pour standardiser les technologies du Web (langages HTML et CSS par exemple).

Les langages HTML et CSS

Les langages HTML et CSS séparent le fond de la forme :
  • HTML permet:
    • de créer des hyperliens,
    • de s'occuper du "fond" du contenu :
      • structuration logique du texte (titres, sous-titres, paragraphes, tableaux, images, ...).
      • inclusion de ressources multimédias (images, vidéos, sons, animations, ...),
    • de mettre un minimum en forme le contenu (police, couleur, encadrer, souligner, mettre en gras, ..).
  • CSS s'occupe de la "forme" en gérant l’esthétique de la page web plus finement que ne le permet le HTML.

Le langage HTML

HTML signifie HyperText Markup Language autrement dit "langage de balisage hypertexte".

Un texte écrit en utilisant ce langage (on parle de "code HTML") s'enregistre dans un fichier d'extension .html. Ce fichier s'ouvre avec un navigateur internet qui en interprète le contenu, il affiche alors la page web définie par le fichier et mène les actions demandées par l'utilisateur (suivi de lien, lecture de vidéo, ...).

Pour indiquer commment agir sur un texte (ou une image, ...), on utilise des balises HTML que l'on écrit en général avant (balise ouvrante) et après (balise fermante) l'élément sur lequel on veut agir.
Par exemple pour mettre en gras du texte on l'encadre par <b> et </b> (il faut indiquer le début et la fin de la mise en gras). Parfois, il n'y a pas de balise de fermeture : pour aller à la ligne on utilise seulement <br> (il n'y a pas de balise fermante car on n'indique que le lieu du retour à la ligne).

Le langage CSS

Le langage CSS ("Cascading Style Sheets" autrement dit "feuilles de style en cascade") permet de mettre en forme une page web bien plus en détail que ne peut le faire le langage HTML.
Il permet de définir le style graphique des pages web (couleur des titres de niveau 1, couleur d'arrière-plan, taille des polices des titres ou des paragraphes, présentation des listes, ...).

Pour faire ceci on écrit du code ressemblant à 

h1 {
  color: red;
  font-size: 5em;
}
h1 indique que l'on définit le style des titres de niveau 1.
Entre les accolades on écrit le code pour définir ce style.
color: et font-size: indiquent que l'on définit la couleur et la taille de police de ces titres.
Après : on trouve les valeurs de ces attributs des titres.
